Migration guide

Move an OnPass user profile from one machine to another.

The exact screen labels can change by version, but the migration flow stays the same: export the user data securely, move the files yourself, and import them on the new machine.

Security note: Exported user files should be treated as sensitive material. Only transfer them through a method you control and delete temporary copies when the migration is complete.

1. Export your user data

In the desktop app, open the export user flow and save the generated files somewhere easy to find, such as a temporary folder on your desktop.

2. Keep the files private

Do not send the exported files through public channels or share them with anyone else. Use a transfer method you control, such as encrypted removable media or a secure private transfer path.

3. Set up OnPass on the new machine

Install or build the desktop app on the destination PC, then open the login screen where import is available.

4. Import the exported user

Select the import user option and point OnPass to the exported files you transferred from the original machine.

5. Sign in and verify your vault

Log in with the imported account credentials, confirm your saved passwords are present, and remove any leftover temporary export files you no longer need.
